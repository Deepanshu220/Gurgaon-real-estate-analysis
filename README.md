# Gurgaon Real Estate Market Analysis

## Project Overview
This project analyzes Gurgaon residential property data using Exploratory Data Analysis (EDA).

## Business Questions
- Which is the costliest flat?
- Which locality has the highest average price?
- Which locality has the highest rate per square foot?
- Do ready-to-move properties cost more?
- Do RERA-approved properties command a premium?
- How does area impact property price?
- Which BHK configuration is the most expensive?
- Which property type is the costliest?
- Which builders consistently price higher?
- Are larger homes more expensive per square foot?

## Solution Overview
We will solve this problem using Exploratory Data Analysis (EDA).
EDA helps us understand patterns, relationships, and anomalies in the data before applying any advanced modeling.
The solution is divided into logical steps:
- Data loading and understanding
- Data cleaning and preparation
- Analysis to answer each business question
- Summary of insights

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
Gurgaon Real Estate Dataset from Kaggle.
